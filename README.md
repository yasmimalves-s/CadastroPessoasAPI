# CadastroPessoasAPI

API CRUD de cadastro de pessoas desenvolvida com ASP.NET Core e Entity Framework.

## Tecnologias utilizadas
- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server
- Swagger

## Funcionalidades
- Listar pessoas
- Buscar pessoa por ID
- Cadastrar pessoa
- Atualizar pessoa
- Excluir pessoa

## Como executar

1. Clone o repositório:

```bash
git clone URL_DO_SEU_REPOSITORIO
```

2. Execute:

```bash
dotnet run
```

3. Abra:

```text
http://localhost:5034/swagger
```
